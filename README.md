## Instalar proyecto (Frontend)

### Clonar repositorio

```bash
# clone the repo
$ git clone https://github.com/oeneika/Tesis-Back.git my-project

# go into app's directory
$ cd my-project

# install app's dependencies
$ npm install
```

## Usos

```bash
# serve with hot reload at localhost:8000.
$ npm start
```
